5. Метод componentDidMount считается самым удачным местом для первого запроса данных с сервера. 
useEffect() - аналогичен componentDidMount
6. Fetch API не будет автоматически генерировать ошибку в случае кода ответа 401. 
Запрос будет успешно выполнен, и вернется объект Response с кодом ответа 401.
Нужно использовать статус ответа для проверки, и вручную генерировать ошибку.
7. Кроме Fetch запросы можно отправлять при помощи различных библиотек, например axios, ky, got.
8. CORS нужен для для безопасной обработки и передачи ресурсов между веб-приложениями на разных доменных источниках. 
Он предоставляет механизмы, которые позволяют серверам указывать, какие источники имеют доступ к определенным ресурсам.
